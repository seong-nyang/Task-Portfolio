## 🛠 사용 기술

- **HTML5**: 시맨틱 태그 활용 (`header`, `nav`, `section`, `footer`)

- **JavaScript**: 인터랙션 기능 (버튼 클릭 시 팝업, 실시간 시간 표시)
- **GitHub Pages**: 정적 웹사이트 배포
